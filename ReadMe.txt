<h2>Como usar</h2>
O atalho executa o programa no cmd. 
Caso o caminho do arquivo esteja errado, edite o "Destino" abrindo as "propriedades" do atalho.
Caso o caminho de seu executável do prompt de comando não seja C:/Windows/System32/cmd.exe altere o caminho do arquivo.
O comando /k presente no "Destino" do atalho server para executar um código no prompt de comando.
os comandos cd .. retornam para o caminho anterior ao caminho atual. Foi usado duas vezes para chegar na raíz do disco.
À partir do cd Users, o caminho pode ser diferente, mas o final sempre será /ProjetoInvent-main/bin & java aplicativo/Main
